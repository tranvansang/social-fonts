Brand Description

About The Brand

Last.fm is a website that uses a music recommendation system called "Audioscrobbler". Last.fm creates a detailed profile of every user's musical taste by keeping track of details like the songs the user listens to, either from Internet radio stations, or the user's computer or many portable music devices. All the information gets transferred to Last.fm's database by the music player (Spotify, Deezer, Clementine, Amarok, MusicBee), or by a plugin installed in the user's music player. The data is then shown on the user's profile page and compiled to create reference pages for individual artists.

What font does Last.fm use?

All the characters in Last.fm's logo use the font The National Medium except for the letter "a" which uses custom design. You can view, test with the font generator, and download the font here.
