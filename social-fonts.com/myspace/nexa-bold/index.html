Brand Description

About The Brand

Myspace is a social network with an interactive, user-submitted network of friends, user profiles, blogs, groups, photos, music, and videos. The MySpace headquarters is located in Beverly Hills, California.

What font does MySpace use?

The font used in the MySpace logo is called Nexa Bold. You can view, test with the font generator, and download the font here.

The previous MySpace logo, which has been obsolete since 2010, uses the font Ariel Rounded Bold.
