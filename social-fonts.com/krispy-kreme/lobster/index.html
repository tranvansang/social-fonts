Brand Description

About The Brand

Krispy Kreme is an American global doughnut company and coffeehouse chain based in Winston-Salem, North Carolina and founded by Vernon Rudolph in 1937.

What font does Krispy Kreme use?

Krispy Kreme’s logo uses two different fonts. The part that reads "Krispy Kreme" was custom drawn but the font Lobster looks very similar to it. The bottom part of the logo reading “Doughnuts”, uses the font Futura Bold by Paul Renner. You can test out the Lobster font with the font generator.
